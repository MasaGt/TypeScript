### 型アノテーション、型推測、型アサーションとは

#### 型アノテーション (type annotation)

- variable: datatype で変数の宣言時に代入できるデータ型を定義すること

    ```ts
    let num: number = 123;

    // 特定のプロパティを持つオブジェクト型の変数を宣言
    let obj: {name: string};
    obj = {
        name: "Andy"
    };

    // 関数型の変数も宣言可能
    let func: (arg: number) => string;
    func = (arg) => {
        return `${arg}`;
    }
    ```

<br>

#### 型推測 (type inference)

- 変数のデータ型が代入された値によってコンパイラに推測されること

    ```ts
    // 変数strはコンパイラによってstringと推測される
    let str = "hello";
    // よって、string型で使えるプロパティや関数を呼び出すことができる
    str.length;
    ```

<br>

#### 型アサーション (type assertion)

- as などによってコンパイラによって分析されたデータ型を上書きすること

- バグの温床になるので、基本的には使わないほうがいい

    ```ts
    let val: string | number = "string";

    // valはstring型であることをコンパイラに伝える
    let len = (val as string).length;
    ```
